# AI 学习路线

## 目标

- 建立模型、Prompt、Agent 和工具实践的长期知识库。
- 每个主题都保留概念、案例、问题、实践结果和下一步行动。
- 优先记录自己真实用过的工作流，避免只收藏链接。

## 栏目

| 栏目 | 内容 | 产物 |
| --- | --- | --- |
| 模型笔记 | 模型能力、限制、价格、适用场景 | 对比表、选型记录 |
| Prompt | 研究、写作、代码、复盘提示词 | 可复用模板 |
| Agent | 多步骤任务、工具调用、自动化流程 | 工作流说明 |
| 论文文章 | 论文、技术博客、行业文章 | 摘要和问题清单 |
| 项目实践 | 小工具、数据处理、网页应用 | Demo 和复盘 |
