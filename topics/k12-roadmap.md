# 学科学习路线

## 目标

- 建立小学、初中两个学段的长期知识库。
- 每个学科都保留知识点、例题、错题、复习记录和下一步行动。
- 优先沉淀真实学习中遇到的问题，而不是只做目录收藏。

## 小学学科

| 学科 | 内容 | 产物 |
| --- | --- | --- |
| 语文 | 识字写字、阅读理解、古诗文、习作表达 | 阅读卡、作文素材、古诗文卡 |
| 数学 | 数与运算、图形、计量、应用题、数学思维 | 知识点卡、例题、错题复盘 |
| 英语 | 自然拼读、词汇句型、听说读写 | 单词卡、句型卡、听读记录 |
| 科学 | 生命、物质、地球宇宙、实验观察 | 观察记录、实验卡、概念图 |
| 道德与法治 | 生活规则、社会常识、责任意识、法治启蒙 | 主题卡、情境讨论 |
| 信息科技 | 计算思维、数字工具、基础编程、信息安全 | 操作记录、小项目、概念卡 |

## 初中学科

| 学科 | 内容 | 产物 |
| --- | --- | --- |
| 语文 | 现代文、文言文、古诗词、名著阅读、作文 | 阅读分析、文言词卡、作文素材 |
| 数学 | 代数、几何、函数、统计概率、综合题 | 公式卡、题型卡、错题本 |
| 英语 | 词汇语法、阅读、听力、写作、口语 | 语法卡、阅读笔记、写作模板 |
| 物理 | 力、热、光、电、声、实验探究 | 概念卡、实验卡、模型图 |
| 化学 | 物质构成、化学反应、实验操作、计算 | 方程式卡、实验卡、计算题型 |
| 生物 | 细胞、人体、生物圈、遗传进化、实验观察 | 概念图、结构图、实验记录 |
| 历史 | 中国史、世界史、时间线、人物事件、因果分析 | 时间线、人物卡、事件复盘 |
| 地理 | 地图、地球、区域地理、自然与人文环境 | 地图卡、区域卡、图表分析 |
| 道德与法治 | 个人成长、社会生活、法治常识、国情理解 | 主题卡、材料分析、观点提纲 |

## 记录模板

| 字段 | 说明 |
| --- | --- |
| 学段 | 小学或初中 |
| 学科 | 语文、数学、英语等 |
| 知识点 | 本次学习的最小主题 |
| 例题/材料 | 对应题目、文章、实验或情境 |
| 易错点 | 做错、看不懂或容易混淆的地方 |
| 复习动作 | 下一次要做的练习、背诵或讲解 |
